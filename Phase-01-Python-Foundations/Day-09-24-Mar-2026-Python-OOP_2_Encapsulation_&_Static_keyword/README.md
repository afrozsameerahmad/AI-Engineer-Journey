1. How objects access attributes
2. What happens when an attribute does not exist
3. Creating attributes outside the class
4. Reference variables and multiple references to same object

7. Static variable vs instance variable

# Exercise Notebooks

1. `exercises/encapsulation.ipynb`  
   Covers object attribute access, missing attributes, dynamic attribute
   creation, reference variables, and passing objects to functions.

2. `exercises/static_variables.ipynb`  
   Uses ATM class example to explain static variables and the difference between
   class-level and object-level data.

# Practice Notebook (`practice/`)
