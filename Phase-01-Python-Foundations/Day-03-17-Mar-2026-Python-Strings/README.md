# Concepts Practiced

## String Basics

- Single, double, and triple-quoted strings
- Unicode string concept
- Using quotes correctly inside strings

## Accessing Strings

- Positive indexing
- Negative indexing
- Slicing with `start:end:step`
- Reversing strings with slicing

## String Behavior

- Strings are immutable
- `del` removes the variable reference

## String Operations

- Concatenation
- Repetition
- Comparison
- Logical operations
- Looping through strings
- Membership checking

## String Functions and Methods

- `len()`, `max()`, `min()`, `sorted()`
- `upper()`, `lower()`, `title()`, `capitalize()`, `swapcase()`
- `count()`, `find()`, `index()`
- `startswith()`, `endswith()`
- `format()`
- `isalpha()`, `isdigit()`, `isalnum()`, `isidentifier()`
- `split()`, `join()`, `replace()`, `strip()`
