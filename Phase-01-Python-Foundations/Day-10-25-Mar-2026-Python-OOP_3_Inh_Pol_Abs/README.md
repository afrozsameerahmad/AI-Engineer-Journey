# Day 10 - Python OOP 3 (Inheritance, Polymorphism, Abstraction, Aggregation)

**Date:** 25 March 2026

This folder contains Day 10 work focused on core OOP design concepts:
inheritance patterns, polymorphism behavior, abstract classes, and aggregation.

---

# Day 10 Coverage

1. Inheritance basics and real use-case idea
2. What gets inherited in child classes
3. Private vs non-private members in inheritance
4. `super()` keyword usage in methods and constructors
5. Types of inheritance:
	- Single
	- Multilevel
	- Hierarchical
	- Multiple
	- Hybrid
6. Polymorphism:
	- Simulated method overloading
	- Method overriding
	- Operator overloading
7. Abstraction using `ABC` and `@abstractmethod`
8. Aggregation (Has-A relationship)

Notebook focus summary:

1. `inheritance.ipynb`
	Covered parent-child relationship, inherited constructor/method behavior,
	private member access rules, and multiple inheritance types.

2. `polymorphism.ipynb`
	Covered polymorphism forms with examples for method overriding, simulated
	overloading, and operator overloading with `__add__`.

3. `abstraction.ipynb`
	Covered abstract class design using `BankAccount` and concrete classes
	`WebBankAccount` and `MobileBankAccount`.

4. `aggregation.ipynb`
	Covered Has-A relationship with `Customer` and `Address` classes and
	practical object collaboration.
