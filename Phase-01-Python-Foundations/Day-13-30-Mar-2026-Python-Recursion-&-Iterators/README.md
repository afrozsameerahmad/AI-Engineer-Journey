# Day 13 - Python Recursion 
**Date:** 30 March 2026

This folder contains Day 13 work focused on solving problems using recursion and understanding its trade-offs compared to iteration.

---

# Day 13 Coverage

1.  **Recursion Fundamentals**:
    -   Definition of a recursive function.
    -   Base case vs. recursive case.
    -   The function call stack.
    -   Recurrence relations.
2.  **Classic Recursive Problems**:
    -   Calculating factorial.
    -   Summing the first N natural numbers.
3.  **Recursive List Traversal**:
    -   Printing all elements of a list recursively.
4.  **Creative Recursive Solutions**:
    -   Implementing multiplication through recursive addition.
    -   Checking if a string is a palindrome.
5.  **Recursion vs. Iteration**:
    -   Analyzing the Fibonacci sequence (Rabbit Problem) to compare the efficiency of a recursive vs. an iterative solution.
6.  **Advanced Recursion: Backtracking**:
    -   An introduction to backtracking to generate the power set of a given set.

# Exercise & Task Notebooks

1.  `exercises/recursion.ipynb`
    -   Introduced the core concepts of recursion using the factorial function.
    -   Defined base case, recursive case, and call stack.

2.  `tasks/cal_n_natural_num.ipynb`
    -   Practiced writing a simple recursive function to find the sum of numbers up to `n`.

3.  `tasks/list.ipynb`
    -   Implemented a recursive function to iterate through a list and print its elements.

4.  `tasks/multiply.ipynb`
    -   Solved multiplication by using only addition in a recursive manner.

5.  `tasks/palindrome.ipynb`
    -   Wrote a recursive function that checks if a string reads the same forwards and backwards.

6.  `tasks/rabbit_problem.ipynb`
    -   Modeled the Fibonacci sequence to solve the rabbit population problem.
    -   Used an efficient iterative approach to highlight performance differences with deep recursion.

7.  `tasks/power_set.ipynb`
    -   Explored backtracking, a powerful recursive technique, to find all possible subsets of a set.
