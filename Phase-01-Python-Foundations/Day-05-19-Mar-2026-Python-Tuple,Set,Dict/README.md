# Concepts Practiced

## Tuple

- Tuple creation: empty, single-item, mixed
- Indexing and slicing
- Tuple methods: `count()`, `index()`
- Packing/unpacking and `zip()`/unzip
- Immutability behavior and tuple as dictionary key
- Tuple vs list quick comparison

## Set

- Set creation and uniqueness
- Add/update/remove/discard/pop/clear
- Set operations: union, intersection, difference, symmetric difference
- Membership checks and subset/superset/disjoint
- Set comprehension
- `frozenset` as immutable set

## Dictionary

- Create, access, update, delete key-value pairs
- Safe access using `get()`
- Iteration with keys, values, items
- Dictionary comprehension and inversion
- Frequency counting pattern
- Merging dictionaries (`update`, unpacking)
