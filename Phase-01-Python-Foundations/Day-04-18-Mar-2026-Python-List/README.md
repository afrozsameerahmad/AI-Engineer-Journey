# Concepts Practiced

## List Basics

- Creating lists (empty, 1D, 2D, 3D)
- Heterogeneous and homogeneous lists
- Using type conversion to create lists

## Accessing Lists

- Positive and negative indexing
- Slicing with `start:end:step`
- Reversing lists with slicing

## List Behavior

- Lists are mutable
- Lists allow duplicates and mixed types

## List Operations

- Append, insert, extend
- Concatenation and repetition
- List comprehension
- Editing and deleting items

## List Functions and Methods

- `len()`, `sum()`, `max()`, `min()`, `sorted()`, `reversed()`
- `count()`, `index()`, `sort()`, `copy()`
