
# Day 19: Selection, Bubble, and Insertion Sort

## Overview
This day covers three fundamental sorting algorithms:

- **Bubble Sort**: Repeatedly compares and swaps adjacent elements to sort the list.
- **Insertion Sort**: Builds a sorted list one element at a time by inserting elements into their correct position.
- **Selection Sort**: Selects the minimum (or maximum) element from the unsorted part and swaps it with the first unsorted element.
