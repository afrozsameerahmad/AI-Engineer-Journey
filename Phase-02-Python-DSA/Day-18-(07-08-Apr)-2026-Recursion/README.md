# Day 18: Recursion

## Overview
This day focuses on understanding and applying recursion in Python to solve classic problems.

- What is recursion?
- Types of recursion: functional and parameterized
- Base case and recursive case
- Recursion vs iteration
- Time and space complexity analysis
- Practice with classic recursion problems

## Folder Structure & Practice Files

- `exercise/func_rec.ipynb`  
	- Functional and parameterized recursion examples
- `exercise/rec_pparameters.ipynb`  
	- Recursion with parameters, printing values recursively

- `practice/fact.ipynb`  
	- Factorial using recursion
- `practice/fib.ipynb`  
	- Fibonacci sequence using recursion
- `practice/palindrome.ipynb`  
	- Palindrome check (loop and recursion)
- `practice/rev_array.ipynb`  
	- Reverse an array using recursion

## Learning Objectives
- Understand the concept and structure of recursion
- Identify and write base and recursive cases
- Implement both functional and parameterized recursion
- Analyze time and space complexity of recursive solutions
- Practice recursion with classic problems (factorial, Fibonacci, palindrome, array reversal)
