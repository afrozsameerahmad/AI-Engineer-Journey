# Day 23: LeetCode Question Practice

## Overview
This day is based on three notebook implementations focused on array practice:

- Linear Search
- Move Zeroes to End
- Merge Two Sorted Arrays (with duplicate handling in output)

These notebooks practice linear scan, in-place array modification, and two-pointer traversal.

## Notebook Files
- `linear_search.ipynb`: Linear search with index print on match and complexity comments
- `move_zeros_in_end.ipynb`: Brute force and optimized (in-place swap) approaches
- `merge_2_sorted_array.ipynb`: Merge two sorted arrays and skip duplicates while appending

## What Was Practiced
- Sequential search in unsorted arrays
- Moving zeros to end while preserving non-zero order
- Using two pointers for sorted array merge
- Writing time and space complexity after each solution

## Complexity Snapshot (As Written in Notebooks)

| Problem | Approach | Time | Space |
|---|---|---|---|
| Linear Search | Single scan | O(n) | O(1) |
| Move Zeroes | Brute force (temp array) | O(n) | O(n) |
| Move Zeroes | Two pointers (in-place) | O(n) | O(1) |
| Merge Two Sorted Arrays | Two pointers + result array | O(n + m) | O(n + m) / O(1)* |

*The notebook notes mention O(1) when output array space is not counted.

## Key Patterns Practiced
1. Linear Scan
2. Two Pointers
3. In-place Swapping
4. Sorted Array Traversal

## Difficulty Level
Easy to Easy-Medium. These are high-frequency interview patterns.
